Intro

Installable via npm install --save peer-book, it can also be used directly in the browser.

Download

The source is available for download from GitHub. Alternatively, you can install using npm:

$ npm install --save peer-book

You can then require() peer-book as normal:

const peerBook = require('peer-book')

In the Browser

Peer-book should work in any ES2015 environment out of the box.

Usage:

<script type="text/javascript" src="index.js"></script>

The portable versions of peer-book, including index.js and index.min.js, are included in the /dist folder. Peer-book can also be found on unpkg.com under

  • https://unpkg.com/peer-book/dist/index.min.js
  • https://unpkg.com/peer-book/dist/index.js
